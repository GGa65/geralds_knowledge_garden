# Blogging Task Listen ohne weitere Gruppierung

**Codierung**:
```
```tasks
not done
path includes Learnings
```


**Ausgabe**
```tasks
not done
path includes Learnings
```


# Blogging Task Listen nach Pfad gruppiert

**Codierung**
```
```tasks
not done
path includes Learnings 
group by path
```


**Ausgabe**
Nach Pfad gruppiert rendert die Rückgabe des Codes übersichtlicher 
```tasks
not done
path includes Learnings 
group by path
```
